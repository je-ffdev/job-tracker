# 📋 Job Application Tracker

A clean, fast, browser-based job application tracker. No backend, no login, no fuss — just open and start tracking.

Built by [@je-ffdev](https://github.com/je-ffdev) during a late-night build session.

## Features

- **Add & edit applications** — company, role, status, date, location, salary, notes, job URL
- **5 status stages** — Wishlist → Applied → Interview → Offer → Rejected
- **Live stats bar** — total, per-status counts, and response rate
- **Filter & search** — by status or by company/role name
- **Colour-coded cards** — instant visual overview of your pipeline
- **Keyboard shortcut** — press `N` to add a new application instantly
- **Persistent storage** — everything saved in `localStorage`, survives refreshes
- **Zero dependencies** — pure HTML, CSS, JavaScript

## Usage

Just open `index.html` in any browser. That's it.

Or visit the live version: [je-ffdev.github.io/job-tracker](https://je-ffdev.github.io/job-tracker)

## Deploying to GitHub Pages

1. Push this repo to GitHub
2. Go to **Settings → Pages**
3. Set source to `main` branch, `/ (root)`
4. Done — live at `https://je-ffdev.github.io/job-tracker`

## Tech Stack

- HTML5 / CSS3 / Vanilla JavaScript
- Google Fonts (Playfair Display, DM Mono, Outfit)
- localStorage for persistence
